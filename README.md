# Fraud Detection Project

This is a simple machine learning project to detect fraudulent transactions.

The model learns from data and tries to classify each transaction as:
- Fraud
- Not Fraud

---

## 📌 What this project does

- Loads and explores the dataset  
- Cleans and prepares the data  
- Trains a machine learning model  
- Evaluates the model performance  

---

## 🧠 Tools used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

---

## ⚙️ Steps

1. Data preprocessing  
2. Data analysis (EDA)  
3. Model training  
4. Model evaluation  

---

## 📊 Challenges

- Data is imbalanced (few fraud cases)  
- Need to focus on recall and precision
